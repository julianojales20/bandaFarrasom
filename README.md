# Banda Farrasom

Site de propostas da Banda Farrasom — React + Vite.

## Desenvolvimento

```powershell
npm install
npm run dev
```

Abra `http://localhost:5173` no navegador.

## Produção

```powershell
npm run build
```

A saída fica em `dist/`. O projeto está configurado para deploy na Vercel (`vercel.json`).
